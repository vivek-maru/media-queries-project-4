# media-queries-project-4

Structure and Layout
Header & Navigation: Features a dark theme navigation bar containing the brand name ("COFFEE SOLUTIONS") and menu links (Home, Menu, Blog, Media, Contact).

Hero Section: Includes a large title ("FRESHLY ROASTED COFFEE"), a call-to-action button ("Shop now"), and a circular image of a coffee cup.

Curved Wave Background: A CSS-styled wave shape separates the dark header area from the light body content below, mimicking the design in the image.

Sections: The page is organized into three main content sections: "TOP CATEGORIES," "TOP MILK SHAKES," and "LATEST BLOGS".

Design and Styling
Color Palette: The site uses a beige background (#e5c9b8) for the main body and a dark theme (#1a1a1a) for the header and footer.

Cards: Content is structured in cards with rounded corners (border-radius: 20px).

Categories: Feature background images with a dark overlay text.

Shakes: Display an image, a "likes" count, and a "BUY NOW" button.

Blogs: Contain a featured image, a title, a short description, and a "READ MORE" button.

Typography: The layout uses a clean sans-serif font (Segoe UI) for readability.

Interactivity and Responsiveness
Responsiveness: The layout is mobile-first, stacking items vertically on small screens and switching to a horizontal 3-column layout (flex-direction: row) on desktop screens via media queries (@media (min-width: 768px)).

Flexbox and Grid: The code heavily utilizes CSS Flexbox for aligning items and CSS Grid for organizing the footer columns.

Font Awesome: Icons are used for search, shopping carts, social media links, and likes, loaded via an external Font Awesome library.

Footer
Full Footer: The footer is structured into four columns: Products, Category, Company Info, and Social Links.

Bottom Bar: Contains links for privacy, cookie settings, and legal information.
